Basic Fundamentals

The Renowned Web

The web is basically a group of computers that interacts with each other. When an individual opens up a web page (Yahoo! or Google), their computer puts out an HTTP request to an designated server. What the server does next is locate the required HTML document which it transfers back to the user's computer via an web browser (Internet Explorer or Firefox) that shows up on the computer screen.

H.T.M.L

HTML is widely used in virtually every computer concept on the web. This Hypertext Markup Language is most verstaile to write any programming code. Other languages exist such as Javascript, Python, Ruby, and so on.

Tags/Elements

Within HTML, there are elements & tags. Elements are groups of code inputted while tags mark an element with a function such a name or command. Two element types was mentioned: inline and block. Inline breaks off
and forms a new line of code. Block surrounds the content with an invisible barrier.

"Dumb" Tech

Computers read and executes all commands with precision. If a programmer inputs specific directions to make a computer randomize numbers or scan files, the program would run effective. However, if one step is misplaced, the program will fail to work. Programmers label the computers as "stupid", but we human beings create the technology so the fault is pretty much shared on both sides.
